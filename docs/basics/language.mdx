### How to change the language?

To change the language of KyuBot, you can use the `/language` command and select one of the available languages. We are working on adding more languages to KyuBot, so if you want to help us translate KyuBot to your language, you can do it by following the steps below.

### How to translate KyuBot?

If you want to translate KyuBot to your language, you can do it by following the steps below.

1. Go to the [KyuBot Translations](https://github.com/KyuManagement/KyuTranslation) repository.
2. Clone the repository using the `Fork` button on the top right corner.
3. Create a new folder in the `languages` folder using the locale from the [Discord API](https://discord.com/developers/docs/reference#locales).
4. Copy the files from `languages/en-GB` and paste them in the new folder you created.
5. Translate the contents of the files.
6. Create a pull request using your fork.
7. Wait for the pull request to be merged.
8. Thank you for contributing to KyuBot!
