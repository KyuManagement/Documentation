### What does KyuBot do? ü§î

KyuBot is a Discord bot that focuses on creating the best leveling experience for your community. It is currently one of the most indepth leveling bots on Discord, with many features that are not found in other bots. KyuBot is constantly being updated with new features and improvements, and we are always looking for feedback and suggestions to improve the bot.

### Feedback / Reporting Bugs üêû

If you have any issues, feedback, or suggestions for KyuBot, join the <Highlight color="#69d9a1" url="https://discord.com/invite/bnkaWnwuVV">support server</Highlight> to let us know! We are always looking to improve our service. We are always working on new features and improvements to the bot, so if you have any suggestions, please let us know!
